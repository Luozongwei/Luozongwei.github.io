📢📢📢安装go的SDK

go官网：The Go Programming Language下载想要的SDK版本（需要科学上网）

🌻🌻go SDK 的环境设置

📣GOROOT

🏅GOPATH （选弄）

GOPATH选弄的原因在：golang调用本地包报错原因及解决方案

用于存放以后go用到的依赖

🍉配置go SDK (选弄)

windows+R+cmd 输入下面两条指令

go env -w G0111MODULE=on
go env -w GOPROXY=https://goproxy.cn,direct

如果上面两条指令执行失败换下面两条指令：

set G0111MODULE=on

set GOPROXY=https://goproxy.cn,direct

💜💜💜goland创建项目

一定要选择Go。原因：对于goland相对较新一些版本新建项目时没有go modi模式选项的坑

这个go.mod要有 module 和 go版本就行

这里建议选择go 1.16以上，因为在1.16以上才有debug功能

添加一个插件可以进行自动保存 相当于idea: crlt + alt + L

配置FIle Watcher后就会保存.go文件了

如果项目跑不起来的话将Edit Cofigurations中的Run kind改为file

❤️❤️❤️忙碌的敲代码也不要忘了浪漫鸭！
