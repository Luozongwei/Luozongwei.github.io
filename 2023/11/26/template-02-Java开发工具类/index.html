前言

俗话说得好：学渣就是一个字，抄！

本文章主要是将日常学习中用到的Java工具类模板记录下来。

JWT

  • maven
  • annotation
    • @WebFilter(urlPatterns = “/*”)
    • @ServletComponentScan： 启动类上添加
  • JwtUtil

WebConfig

  • 跨域问题
  • 拦截器
  • JwtInterceptor

统一响应实体

  • Result

统一响应格式

  • ResultCode

全局异常处理

  • GlobalExceptionHandler
  • ServiceException

mybatisPlus代码生成器

  • maven
  • MybatisPlusGenerator

Knife4jConfig

md5

主要用户密码或者敏感信息加密

ExcelUtils

对excel基本导入导出功能的封装，比如导出一个sheet和导出多个sheet。

但是对于复杂excel的导入导出还需要自己设计

mybatis-plus代码生成器

  • mavne
  • MybatisPlusGenerator
  • mp分页功能开启

IP工具类
